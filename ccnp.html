// exam 1 · encor 350-401 · v9 curriculum

ENCOR — Core Enterprise Networking (v9)

DOMAIN 1 · 15%Architecture
  • Enterprise network design — 2-tier vs 3-tier vs spine/leaf
  • SD-Access fabric — underlay, overlay, control/data plane
  • SD-WAN architecture — components and traffic flow
  • Cloud on-ramp — colocation, IaaS, SaaS
  • High availability — graceful restart, NSF, NSR
DOMAIN 2 · 10%Virtualization
  • Network virtualization — VRF, GRE, LISP
  • Data center virtualization — VXLAN concepts, NV overlay
  • Virtualization components — hypervisors, vSphere, KVM
  • Container networking — Docker, Kubernetes basics
DOMAIN 3 · 30%Infrastructure
  • Layer 2 — advanced STP (MSTP), PVST+ tuning, BPDU guard/filter
  • Layer 3 — OSPFv2/v3 multi-area, LSA types 1–7, area types
  • BGP — eBGP/iBGP, route reflectors, attributes, path selection
  • EIGRP — named mode, dual algorithm, feasible successor
  • Multicast — PIM-SM/DM, IGMP, RP placement, BSR
  • QoS — classification, marking, queuing, shaping, policing, DSCP
  • IP services — FHRP (HSRP/VRRP/GLBP), NAT/PAT, DHCP relay
DOMAIN 4 · 25%Network Assurance
  • Troubleshooting methodology — OSI-based structured approach
  • SNMP v2c/v3, NetFlow/IPFIX, SPAN/RSPAN/ERSPAN
  • IP SLA — probes, reaction, tracking objects
  • Cisco DNA Center — assurance, telemetry, path trace
  • Wireshark and CLI debug techniques
DOMAIN 5 · 20%Security
  • Device access — AAA, TACACS+, RADIUS, local fallback
  • Control plane policing (CoPP)
  • Zone-based firewall (ZBFW) on IOS
  • MACsec — 802.1AE layer 2 encryption
  • 802.1X — port-based authentication, EAP types
  • uRPF, IP source guard, DHCP snooping advanced
  • VPN — DMVPN (phase 1/2/3), FlexVPN overview
DOMAIN 6 · 10%Automation
  • Python for networking — Netmiko, NAPALM, nornir
  • Ansible — playbooks, roles, IOS-XE modules
  • REST APIs — IOS-XE RESTCONF, DNA Center API
  • YANG models — pyang, NETCONF operations
  • Git — version control for network config
  • CI/CD pipeline concepts for network automation
// exam 2 · enarsi 300-410 · v9 curriculum

ENARSI — Advanced Routing & Services (v9)

DOMAIN 1 · 35%Layer 3 Technologies
  • OSPFv2/v3 advanced — virtual links, stub/NSSA areas, filtering
  • EIGRP advanced — summarization, filtering, named mode
  • BGP advanced — communities, local preference, MED, weight
  • BGP route policies — prefix lists, route-maps, regexp
  • Path manipulation — AS-path prepending, conditional advertisement
  • IPv6 routing — redistribution, filtering, dual-stack design
  • Route redistribution — multiple protocols, metric translation
DOMAIN 2 · 20%VPN Technologies
  • MPLS — LDP, label assignment, label switching
  • MPLS L3VPN — MP-BGP, VRF, route distinguisher/target
  • DMVPN — phase 1, 2, 3 — mGRE, NHRP, IPSec profile
  • FlexVPN — IKEv2, smart defaults, spoke-to-spoke
  • IPSec — IKEv1/v2, transform sets, crypto maps vs VTI
DOMAIN 3 · 25%Infrastructure Security
  • Device hardening — AAA, privilege levels, role-based access
  • Control plane — CoPP, CPPr, routing protocol authentication
  • IPv6 security — RA guard, DHCPv6 guard, SEND
  • ZBFW advanced — policy maps, inspect, URL filtering
  • uRPF strict vs loose mode
DOMAIN 4 · 20%Infrastructure Services
  • IP SLA — advanced probes, tracking, floating static routes
  • Object tracking — interface, route, IP SLA combined
  • NetFlow v5/v9, flexible NetFlow, IPFIX
  • SNMP advanced — MIB walks, trap configuration
  • BFD — Bidirectional Forwarding Detection with OSPF/BGP
  • Performance routing (PfR/OER) concepts
