Full Course Syllabus

Aligned with Cisco's official CCNA 200-301 exam topics v1.1. All six domains covered in order.

DOMAIN 1 · 20%Network Fundamentals
  • Network components: routers, switches, firewalls, APs, controllers
  • TCP/IP model vs OSI — layer functions and PDUs
  • IPv4 addressing — classful, CIDR, VLSM, subnetting
  • IPv6 addressing — types, EUI-64, stateless autoconfiguration
  • TCP vs UDP — ports, three-way handshake, windowing
  • Ethernet switching — MAC table, ARP, frame forwarding
  • Wireless concepts — 802.11 standards, frequencies, channels
  • Physical cabling — copper, fiber, SFP, UTP categories
DOMAIN 2 · 20%Network Access (Switching)
  • VLANs — creation, port assignment, 802.1Q trunking, DTP
  • STP — 802.1D, PVST+, port roles and states, BPDU
  • Rapid PVST+ — RSTP port roles, proposal/agreement
  • EtherChannel — LACP, PAgP, configuration and verification
  • Layer 3 switching — SVIs, routed ports, inter-VLAN routing
  • Cisco WLC — CAPWAP, AP modes, WLAN profiles
  • WLAN security — WPA2 Personal/Enterprise, 802.1X basics
DOMAIN 3 · 25%IP Connectivity (Routing)
  • Routing fundamentals — routing table, AD, metric, longest match
  • Static routing — host routes, network routes, default routes
  • OSPFv2 single-area — router-id, DR/BDR election, LSA types
  • OSPFv2 configuration and verification — neighbors, LSDB, SPF
  • First Hop Redundancy — HSRP, VRRP concepts and config
  • IPv6 routing — static IPv6, OSPFv3 concepts
DOMAIN 4 · 15%IP Services
  • NAT — static NAT, dynamic NAT, PAT (overload)
  • DHCP — server config, relay agent, DORA process
  • DNS — A, AAAA, CNAME, MX records; resolution process
  • NTP — stratum, synchronization, NTP server config
  • SNMP — v2c vs v3, MIB, OID, traps vs polling
  • Syslog — severity levels, logging destinations
  • SSH — key generation, VTY config, version 2
  • TFTP/FTP — IOS image management
  • QoS concepts — DSCP, queuing, traffic shaping
DOMAIN 5 · 15%Security Fundamentals
  • Standard and extended ACLs — numbered and named
  • ACL placement — inbound vs outbound, wildcard masks
  • Port security — max MACs, violation modes, sticky
  • DHCP snooping — trusted/untrusted ports, rate limiting
  • Dynamic ARP inspection (DAI)
  • AAA — RADIUS vs TACACS+, authentication models
  • VPN types — site-to-site, remote access, IPSec overview
  • Common threats — DoS, MITM, spoofing, phishing
DOMAIN 6 · 5%Automation & Programmability
  • Network automation benefits — intent-based networking
  • REST APIs — HTTP methods, JSON, status codes
  • Configuration management — Ansible, Puppet, Chef concepts
  • YANG data models — NETCONF, RESTCONF
  • Cisco DNA Center — SD-Access, policy, assurance
  • Traditional vs controller-based networking
