FortiGate Deep-Dive

FG-01FortiOS Architecture & Platform
  • FortiGate hardware models and virtual appliances (VM)
  • FortiOS architecture — NP7, CP9, management plane
  • Packet flow — ingress, kernel space, session table, egress
  • GUI, CLI, FortiManager API administration modes
  • Operating modes — NAT/Route vs Transparent
  • VDOMs — Virtual Domains, inter-VDOM routing
FG-02Security Policy & Objects
  • Firewall policy architecture — source/destination/service/action
  • Address objects, address groups, wildcards, geography
  • Service objects and service groups
  • Explicit deny, implicit deny, logging best practices
  • Policy routing vs interface routing precedence
  • Local-in policy and management access control
FG-03UTM & Security Profiles
  • Antivirus — flow vs proxy mode, FortiSandbox integration
  • Web filter — categories, overrides, URL filter, safe search
  • Application control — App-ID, application overrides
  • IPS — signatures, custom signatures, rate-based detection
  • DNS filter — botnet C2, DNS over HTTPS inspection
  • Email filter — SMTP inspection, FortiMail integration
  • SSL/SSH deep packet inspection — certificate substitution
FG-04NAT Models
  • Central NAT vs Policy NAT — architecture and selection
  • Source NAT — interface NAT, IP pool types (overload, range, fixed-port)
  • Destination NAT — VIPs, server load balancing
  • NAT46 / NAT64 — IPv4-to-IPv6 translation
  • Hairpin NAT — internal server access via public IP
FG-05VPN — IPSec & SSL
  • IPSec IKEv1 / IKEv2 — phase 1 and phase 2 parameters
  • Site-to-site VPN — route-based vs policy-based
  • Dial-up VPN — dynamic peers, XAUTH, certificate auth
  • SSL VPN — web mode and tunnel mode
  • FortiClient integration — EMS, endpoint compliance
  • ADVPN — Autodiscovery VPN for spoke-to-spoke shortcuts
FG-06High Availability & Diagnostics
  • HA modes — Active/Passive and Active/Active
  • Session synchronization and session pickup
  • HA failover testing and heartbeat configuration
  • diagnose debug flow — 7-stage packet trace
  • diagnose sniffer packet — CLI packet capture
  • FortiView, session table analysis, log forensics

Cisco ASA & Firepower FTD

ASA-01ASA Architecture & Policy
  • ASA security levels — inside, outside, DMZ model
  • Access control lists (ACL) — interface-based, global ACL
  • Object and object-group configuration
  • MPF — Modular Policy Framework, class-map, policy-map
  • Connection limits and embryonic limits
  • AAA with RADIUS/TACACS+ for management and cut-through proxy
ASA-02ASA NAT & VPN
  • ASA NAT — Auto NAT vs Manual NAT, NAT order of operations
  • Static NAT, dynamic NAT, PAT, NAT exemption (identity NAT)
  • Site-to-site IKEv1/v2 crypto maps and VTI
  • AnyConnect SSL VPN — clientless and full tunnel
  • ASA HA — Active/Standby failover, state synchronization
FTD-01FTD & FMC Architecture
  • Firepower architecture — Snort engine, Lina (ASA) engine
  • FMC deployment — hardware vs virtual, licensing
  • FTD packet processing order — prefilter vs access control
  • Device registration and health monitoring in FMC
  • Deployment workflow — policy push, change reconciliation
FTD-02FTD Policies & Inspection
  • Access control policy — zones, networks, applications, users
  • Prefilter policy — fastpath, block, analyze
  • IPS policy — Snort rules, variable sets, rule tuning
  • File policy — malware detection, AMP cloud, retrospective
  • SSL policy — decrypt and re-sign, known-key, do-not-decrypt
  • Identity policy — passive identity, AD integration, ISE
